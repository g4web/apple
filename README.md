Тестовое задание на Yii 2 Advanced 
===============================
1) $ git clone git@github.com:g4web/apple.git ./
2) composer update
3) php yii init
4) Отредактируйте файл \common\config\main-local.php
5) php yii migrate
6) Зарегистрировать на frontend'e
7) Войти в backend
8) Перейти в раздел "Яблоня"

Мой код:
-------------------

```
backend
    controllers/
    models/
    services/
    views/
    web/my/
    widgets/  
console
    migrations/
```
